1. Download the zip file

2. Extract the file

3. Install xampp và turn on Apache, MySQL of xampp

4. Install composer 

5. Open PHPMyAdmin (http://localhost/phpmyadmin)

6. Create a database with name cinema

7. Open terminal of project run command: "php artisan migrate"

8. Run command: "php artisan db:seed"

9. Import cinema.sql file(given inside the zip package)

10. Run command: "php artisan serve" 

11. Open link http://127.0.0.1:8000
